Steps to install 'Online MCQ Test' :

1. Install WAMP Server on your network server
2. Copy the 'online_mcq.rar' to 'www' within the WAMP server's installed location
3. Extract the 'online_mcq.rar' using winrar 
4. Import the 'anssy.sql' file from the 'db_backup' folder into your SQL software (wamp>www>online_mcq>db_backup>19042017>anssy.sql)
5. Open your browser (recommended Google Chrome) and enter 'localhost/online_mcq/' in your URL
